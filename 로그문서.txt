  인테리어 시공업체가 고객에게 현장에서 바로 보여줄 수 있는
  3D 공간 설계 + AI 가구 배치 + 견적서 자동 생성 SaaS 서비스입니다.

  핵심 가치:
    "고객 앞에서 3D 공간 시뮬레이션 → AI 추천 가구 배치 → 견적서 즉시 출력"
    → 수주율 향상

  AI 구독 모델: Free / Starter / Pro / Enterprise
  AI 연동: Claude API (가구 배치 추천, 사진 분석, 견적 최적화)

  │  STEP 1. 환경변수 파일 만들기                                           │
  └─────────────────────────────────────────────────────────────────────────┘

  packages/api/ 폴더 안에 .env 파일을 만들고 아래 내용 입력:

    DATABASE_URL=postgresql+asyncpg://sp_user:sp_pass@localhost:5432/spaceplanner
    REDIS_URL=redis://localhost:6379/0
    JWT_SECRET=여기에-랜덤-긴-문자열-입력
    ANTHROPIC_API_KEY=sk-ant-...
    FRONTEND_URL=http://localhost:3000
    ENVIRONMENT=development

  또는 이미 있는 템플릿 복사:
    cp packages/api/.env.example packages/api/.env

  packages/web/ 폴더 안에 .env.local 파일을 만들고 아래 내용 입력:

    NEXT_PUBLIC_API_URL=http://localhost:8000
    NEXT_PUBLIC_APP_URL=http://localhost:3000

  또는 이미 있는 템플릿 복사:
    cp packages/web/.env.local.example packages/web/.env.local


  ┌─────────────────────────────────────────────────────────────────────────┐

  └─────────────────────────────────────────────────────────────────────────┘

    cd "c:/ai프로젝트/3d제안용디자인/files"
    docker-compose up -d

  → PostgreSQL(5432) + Redis(6379) + MinIO(9000) 자동 실행됨
