■ 프로젝트 개요
--------------------------------------------------------------------------------
이름     : SpacePlanner — 인테리어 시공 수주용 3D 공간 설계 + AI 챗봇 SaaS
목적     : 인테리어 업체가 고객에게 3D 공간 시각화 + AI 가구 배치 + 견적서를
           즉석에서 제시하여 수주율 향상
AI 기능  : Claude API 연동 AI 챗봇 (가구 자동 배치, 사진 분석, 견적 최적화)
기술스택 : Turborepo 모노레포
           - 프론트엔드: Next.js 16 + TypeScript + shadcn/ui + Three.js (R3F)
           - 백엔드: FastAPI + Python + PostgreSQL + Redis
           - 엔진: 순수 TypeScript (2D/3D 도면 엔진)
구독모델 : Free / Starter / Pro / Enterprise (AI 사용량 기반)

■ 실행 방법
--------------------------------------------------------------------------------

1. 인프라 실행 (PostgreSQL + Redis + MinIO)
   > cd "c:/ai프로젝트/3d제안용디자인/files"
   > docker-compose up -d

2. Python 가상환경 + 패키지 설치
   > cd packages/api
   > python -m venv venv
   > source venv/bin/activate   (Windows: venv\Scripts\activate)
   > pip install -r requirements.txt

3. DB 마이그레이션 + 시딩
   > alembic upgrade head
   > python scripts/seed_assets.py    (선택: 가구 50개 + 마감재 25개)

4. 개발 서버 실행
   방법 A — Turborepo 한번에
   > npm run dev

   방법 B — 개별 실행
   > 터미널1: cd packages/api && uvicorn app.main:app --reload --port 8000
   > 터미널2: cd packages/web && npm run dev

5. 브라우저 접속
   > http://localhost:3000          (프론트엔드)
   > http://localhost:8000/api/docs (API 문서)
